# SHELL

A simple web-based terminal interface built with PHP.

## About

This project contains two PHP files that provide web-based terminal functionality:

- `bashos.php` - Basic web terminal
- `bashos2.php` - Advanced web terminal with more features

## Usage

1. Upload the PHP files to your web server
2. Open the files in your web browser
3. Use the terminal interface to run commands

## Important

Use responsibly and only on systems you own or have permission to test.
